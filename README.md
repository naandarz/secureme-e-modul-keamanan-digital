**SecureMe** adalah e-modul interaktif berbasis web yang membahas topik **Keamanan dan Privasi Digital**. E-modul ini dirancang untuk membantu mahasiswa memahami cara melindungi data pribadi, mengenali ancaman digital, menghindari phishing, membuat password yang kuat, serta menggunakan internet dengan lebih aman dan bertanggung jawab.

Produk ini dibuat sebagai bahan ajar digital interaktif untuk mendukung pembelajaran mandiri mahasiswa.

## 🎯 Tujuan Pembelajaran

Setelah menggunakan e-modul ini, mahasiswa diharapkan mampu:

1. Menjelaskan konsep keamanan dan privasi digital.
2. Mengidentifikasi ancaman digital seperti phishing, malware, pencurian akun, dan kebocoran data.
3. Menerapkan cara membuat password yang kuat.
4. Memahami pentingnya OTP dan autentikasi dua faktor.
5. Menganalisis contoh kasus pelanggaran privasi digital.
6. Mempraktikkan kebiasaan aman saat menggunakan internet.

## 🚀 Cara Menjalankan Project

### Cara 1: Jalankan Secara Lokal

1. Download atau clone repository ini.
2. Buka folder project.
3. Klik dua kali file `index.html`.
4. E-modul akan terbuka di browser.

### Cara 2: Jalankan dengan Live Server

1. Buka folder project menggunakan Visual Studio Code.
2. Install extension **Live Server**.
3. Klik kanan file `index.html`.
4. Pilih **Open with Live Server**.

---

## 🌍 Cara Deploy ke GitHub Pages

1. Masuk ke repository GitHub.
2. Buka menu **Settings**.
3. Pilih menu **Pages**.
4. Pada bagian **Build and deployment**, pilih:

   * Source: **Deploy from a branch**
   * Branch: **main**
   * Folder: **/root**
5. Klik **Save**.
6. Tunggu beberapa saat sampai link website muncul.

Setelah berhasil, website dapat diakses melalui link GitHub Pages.

## 📌 Topik Materi

### 1. Pengantar Keamanan dan Privasi Digital

Membahas pengertian keamanan digital, privasi digital, serta pentingnya melindungi data pribadi.

### 2. Ancaman Digital

Membahas ancaman seperti phishing, malware, social engineering, pencurian akun, dan kebocoran data.

### 3. Password dan Autentikasi

Membahas cara membuat password yang kuat, bahaya penggunaan password yang sama, OTP, dan autentikasi dua faktor.

### 4. Privasi Data dan Jejak Digital

Membahas data pribadi, izin aplikasi, jejak digital, dan risiko membagikan informasi secara berlebihan.

### 5. Praktik Aman di Dunia Digital

Membahas langkah sederhana untuk menjaga keamanan akun, perangkat, dan aktivitas online.
